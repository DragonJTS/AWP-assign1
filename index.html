The Codings of Jacob Marshall

Writing code that works

My mission is to write code that does what it needs to do, before all else. I am highly passionate about coding, and working with computers. I've been using computers from a very young age, and have always enjoyed expanding my knowledge of them. I believe that the entire basis for a good program or site is a solid back-end, and an attractive front end is nothing if what it's built on isn't ship-shape. As such I love the chance to write code with that in mind, rather then simply say i think that's how it should be.
